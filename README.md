Laravel Portal

A News Portal for demonstration purpose. Based on Laravel 8.0

Requirements

Laravel 8.0




Installation

git clone https://github.com//jbogoeski/newsblog-laravel-tailwindcss.git blog

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan db:seed
